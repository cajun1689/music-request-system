## DJ Bridge

Download the installer for your Mac:

| File | Description |
|------|-------------|
| `.pkg` | **Recommended** — double-click installer, installs to /Applications |
| `.dmg` | Drag-to-Applications disk image |
| `.zip` | Portable zip archive (also used for auto-updates) |

### What's New
- **Serato DJ support** — auto-detect Rekordbox or Serato, or pick one manually
- **Library sync** — upload your full music library so the system knows what you have
- **Play count tracking** — best-match suggestions based on most-played versions
- **Now Playing with AI** — clean track names displayed on the dashboard
- **Auto-updater** — DJ Bridge now checks GitHub for new versions automatically
- **Source ID** — identify which DJ/deck is pushing tracks

### First Launch
1. Open the app from Applications
2. Click the vinyl icon in your menu bar
3. Enter your Event ID and Push Token from the Admin panel
4. Set mode to Auto and you're live!

### Requirements
- macOS 11+ (Apple Silicon or Intel)
- Rekordbox 6/7 or Serato DJ running
